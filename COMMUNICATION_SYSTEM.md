# Radio Operator
## Communication Received
### From enemy captain:
* Moves (not including stealth)
* When a mine is placed (but not where)
* Torpedo explosion (exact coordinates)
* Response to Sonar
* Response to Drone

### From friendly captain:
* Mine activation location
* Torpedo explosion (exact coordinates)

## Communication given
### To Captain:
* Potential enemy location


# Captain
## Communication Received
### From enemy captain
* Enemy moves (not including stealth)
* When a mine is placed (but not where)
* Torpedo explosion (exact coordinates)
* Response to Sonar
* Response to Drone

### From friendly radio operator
* Possible enemy locations

### From friendly first mate
* Which systems are charged and available

### From engineer
* Direction recommendations (to clear systems and such)

## Communication given
### To radio operator
* Ask for location/certainty

### To first mate
* Which system to prioritize charging
* Tell first mate when to use sonar/drone

### To engineer
* Which systems to prioritize keeping available
